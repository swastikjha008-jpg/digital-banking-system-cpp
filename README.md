# Digital Banking System V2

A React-based upgrade of my original C++ banking system project.

This repository started as a simple console-based banking application written in C++. In the new version, I rebuilt the same core banking features using React to create a cleaner and more interactive web interface.

## Project Journey

### Version 1: C++ Console Banking System

The first version of this project was built in C++ as a menu-driven banking program.

It included:
- PIN login
- balance checking
- deposit money
- withdraw money
- transaction saving logic

The goal of the original version was to practice:
- functions
- conditionals
- loops
- file handling
- input validation
- building a real-world style console project

### Version 2: React Banking Interface

In this new version, I converted the same project idea into a React application.

This version keeps the original banking concept but presents it in a more modern and dynamic way.

It includes:
- PIN-based login
- current balance display
- deposit money
- withdraw money
- recent transaction history
- cleaner UI for browser use

## Why I Built V2

I wanted to improve the earlier C++ project by turning it into something more visual and interactive.

This helped me practice:
- React basics
- state management
- handling form inputs
- conditional rendering
- updating UI dynamically
- converting console logic into frontend application flow
